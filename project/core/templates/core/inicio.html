Bienvenidos

Bienvenidos a nuestro sitio web de bienestar, un espacio dedicado a la salud y el equilibrio personal. En este blog, los usuarios pueden registrarse, iniciar sesión y compartir contenido relacionado con el bienestar físico y mental.

La plataforma permite a los usuarios publicar en sus perfiles individuales y en un feed general accesible para todos. Todo el contenido se almacena de manera segura en una base de datos, garantizando la privacidad y la integridad de la información.

Este sitio es el resultado de un proyecto para el curso de Python en CoderHouse.

¡Explora, comparte y sé parte de esta comunidad dedicada a una vida saludable!
